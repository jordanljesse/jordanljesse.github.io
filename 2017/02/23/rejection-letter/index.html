Rejection Letter

Feb 23rd, 2017 9:34 pm

I received my fist rejection letter today from Launch Code after applying for a paid apprenticeship and failing miserable at their HackerRank code puzzles. At first I felt bewildered, as if perhaps I wasn’t smart enough to be a professional programmer. Maybe I just should stick to design, making logos and book covers? Failure is a hard thing sometimes, but I wouldn’t have known about some of my critical week areas(sorting arrays, Math methods) had I not tried and failed at these code puzzles. The result of all this has left me motivated to redouble my efforts and to invest more time in learning my base programming language(Ruby).

I’ve also been working through Ruby Basics on the Odin Project, and today I completed a set of dice, a puppy Tamagotchi-like program, and a simple text based orange tree. They’re all pretty rudimentary, but it’s still exciting to complete these small projects and see that they actually work. It’s much more exciting to get everything working after spending hours not knowing how to solve a problem or being stuck; when it suddenly clicks and things start to work, it’s a great feeling.
